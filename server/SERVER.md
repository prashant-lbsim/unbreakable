# Server
## How to start the server
1. Run **provaMosquito.py**
```
python provaMosquito.py
```
2. Open another terminal and run **server.py**
```
python server.py
```
## How to train data
Run **training.py**
```
python training.py
```
During the training process, **server.py** and **provaMosquito.py** must be killed
